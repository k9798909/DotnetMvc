# DotnetMvc# DotnetMvc 專案

## 介紹

這是一個 .NET 8 的 MVC Web 專案。專案使用 ASP.NET Core MVC ，包含了基本的 CRUD 操作、身分驗證。

## 專案功能

- 使用者認證和註冊
- 基本的 CRUD 操作

## 使用框架

- **.NET 版本**: .NET 8
- **ASP.NET Core**: MVC
- **資料庫**: PostgreSQL
- **前端**: HTML, CSS, JavaScript, JQuery
- **其他**: Docker、Entity Framework、Identity

## 專案啟動
- 安裝docker在資料夾執行docker compose up
